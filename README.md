# Fullstack Todo Application

A fullstack Todo application built with **React**, **Spring Boot**, **PostgreSQL**, and **JWT authentication**.

## Tech Stack
- Frontend: React (Vite)
- Backend: Spring Boot
- Database: PostgreSQL
- Auth: JWT + BCrypt
- ORM: Spring Data JPA

## Features
- User registration & login
- JWT-based authentication
- User-specific todos
- Secure password hashing
- Logout support
